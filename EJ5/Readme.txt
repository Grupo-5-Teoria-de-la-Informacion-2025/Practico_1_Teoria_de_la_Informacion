Manejo de Registros con Longitudes Fijas y Variables

Este script permite guardar y leer registros con dos formatos diferentes: longitud fija y longitud variable.
Los registros contienen datos de personas, como nombre, dirección, DNI y una lista de campos binarios
(como "Estudios primarios", "Estudios universitarios", etc.). Los registros se almacenan en archivos .dat.

Librerías utilizadas

OS: Se utiliza para gestionar rutas de archivos y directorios de manera independiente del sistema operativo. 
Permite obtener la ruta del directorio actual y construir las rutas completas de los archivos donde se guardan y leen los registros.

Instalacion
OS: Se instala al instalar Python (forma parte de la biblioteca estandar de Python).
